# Reverse Linked List

## Description

Given the `head` of a singly linked list, reverse the list, and return _the reversed list_.

**Example 1:**
- _Input_: `head = [1,2,3,4,5]`
- _Output_: `[5,4,3,2,1]`

**Example 2:**
- _Input_: `head = [1,2]`
- _Output_: `[2,1]`

**Example 3:**
- _Input_: `head = []`
- _Output_: `[]`

**Constraints:**

- The number of nodes in the list is the range `[0, 5000]`.
- `-5000 <= Node.val <= 5000`
